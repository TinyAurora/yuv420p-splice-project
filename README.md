# yuv420p-splice-project

### 项目目的
实现 n*n YUV420P 视频图像的拼接

### 项目环境
操作系统：windows 10     
IDE环境：Visual Studio 2017     

### 程序主要功能文件实现如下：     
config.c----------------------配置文件读写     
config.txt--------------------配置文本文件     
log.c-------------------------日志函数文件     
main.c-----------------------主函数逻辑文件     
yuv420psplice.c-------------yuv420p视频拼接功能函数
